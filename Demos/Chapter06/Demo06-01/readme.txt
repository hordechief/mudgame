All of the demos in this book can be compiled by following these procedures:
VC6: open up the .dsw file, and hit compile
VC7: open up the .sln file, and hit compile
GCC: type "make"


This is Demo 6.1, a simple telnet demonstration.
This server, when run, will accept a connection, read characters from a connection,
and print them back out to the client who is connected to it.
Connect to this server by using any telnet client.
